coldrain

发表于2025-03-06|更新于2025-03-08|Loud Rock
|浏览量:

coldrain是拥有日美两国国籍的旋律摇滚乐队。成员有：Ba / RxYxO、Gt / Sugi、Vo / Masato、Dr / Katsuma、Gt / Y.K.C。乐队主要在日本名古屋活动。
2007年组成，以名古屋为中心活动的旋律摇滚乐队，拥有日美两国国籍，极好操纵着旋律丰富多彩和技艺超凡的英文诗式歌，是操纵动和静的技术乐队，互相缠绕的技术经过emoshonaru的成对吉他，空间散发着最强音素，这是来自他们信念的集结，飘荡着的高品质的音符和压倒实况录音的表演，极大显露着国内以及世界以角胜负的潜力。
组成仅仅1个月，就用实况录音会场限制销售1st DEMO CD-R，把名古屋做为中心正式起动的实况录音活动，完成了一度高贵的乐曲和压倒其他实况录音的表演眨眼之间成为人们口中的话题，俯视名古屋的loud锁景色，以首位君临的气势，踏出为今最为稳健的脚步而顺利登顶。
